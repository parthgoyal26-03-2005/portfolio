## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/parthgoyal26-03-2005/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
